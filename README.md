# Expressions

## Описание проекта

Простой учебный проект по работе с математическими выражениями на C++. Библиотека умеет парсить математические формулы, вычислять их значения и брать производные.

Построение AST (абстрактного синтаксического дерева) и вычисление значения выражения.

## Что умеет библиотека

- **Парсинг выражений** — превращает строки типа `"x^2+2*x+1"` в дерево выражений
- **Вычисление** — считает значение выражения с подстановкой переменных
- **Дифференцирование** — берет производные по любой переменной
- **Базовые операции** — +, -, *, /, ^
- **Функции** — sin, cos, ln, exp
- **Упрощение** — пытается упростить выражения после дифференцирования

## Как это работает

Использую паттерн "Visitor" для обхода AST и паттерн "Composite" для представления выражений. Каждое выражение — это узел в дереве, который умеет вычислять себя и брать производную.

Парсер работает по принципу рекурсивного спуска — разбирает выражение по грамматике математических формул.

## Примеры использования

### В коде

```cpp
#include "Expression.hpp"

int main() {
    // Создаем выражение
    Expression expr("x^2 + 2*x + 1");

    // Вычисляем со значением x = 3
    std::map<std::string, long double> vars = {{"x", 3.0}};
    long double result = expr.evaluate(&vars); // = 16

    // Берем производную по x
    Expression derivative = expr.differentiate("x");
    std::cout << derivative.getString(); // "2*x + 2"

    return 0;
}
```

### В командной строке

```bash
# Вычислить выражение
./build/differentiator --eval "x^2+2*x+1" x=3
# 16

# Взять производную
./build/differentiator --diff "x^2+2*x+1" --by x
# 2*x + 2
```

## Что интересного в реализации

- **AST (абстрактное синтаксическое дерево)** — каждое выражение это дерево узлов
- **Рекурсивный парсер** — разбирает выражения по грамматике
- **Паттерны проектирования** — Visitor, Composite, Factory
- **Умные указатели** — использую `shared_ptr` для управления памятью
- **Автоматическое упрощение** — после дифференцирования пытается упростить выражения

## Производительность

- **Парсинг**: O(n) где n — длина выражения
- **Вычисление**: O(n) — обход дерева
- **Дифференцирование**: O(n) — рекурсивный обход с правилами дифференцирования
